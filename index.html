NextScanner does NOT collect, store, or share any personal data.

2. Permission Requirement Notice

NextScanner requires Camera and Storage permissions to function properly. These permissions are essential for scanning documents and saving PDF files.

If these permissions are not granted, core features of the app will not work, and the user may not be able to proceed within the app.

We request these permissions only for essential functionality and not for tracking, analytics, or data collection.

We request permissions strictly for app functionality:

We do not collect user data, so no personal information is processed or shared.
